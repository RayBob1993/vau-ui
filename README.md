# Vau

## Системные требования

* [Node.js](https://nodejs.org/en/download/package-manager) >= 22.17.1
* [pnpm](https://pnpm.io/installation) >= 10.30.3

## Установка

```bash
pnpm add vau
```
```bash
npm install vau
```
```bash
yarn add vau
```

## Локальная разработка

### Установка зависимостей
``` bash
pnpm install
```

### Запуск локального сервера
``` bash
pnpm serve
```

### Создание билда
``` bash
pnpm build
```

## 🪛 Инструменты

### Линтинг кода
``` bash
$ pnpm lint
```

### Тесты
``` bash
$ pnpm test
```
