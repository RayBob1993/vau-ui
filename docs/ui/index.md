# UI

Слой **ui** npm-пакета **`vau`** — готовые виджеты с префиксом **`V`** и точки подключения через **плагины** Vue. Он опирается на примитивы слоя **core** того же пакета и не дублирует нижний слой напрямую в публичном API потребителя.

## Назначение

- Компоненты вида **`V…`** (например регистрация через `app.component`) — согласованный внешний вид и тема.
- **Плагины** — удобная регистрация набора компонентов приложения.
- Зависимость слоёв: **UI → core** допустима; обратная — нет.

## Импорт

```ts
import { VButton, VForm } from 'vau';
```

Точный набор экспортов смотрите в [`src/lib/ui/`](https://github.com/RayBob1993/vau/tree/main/src/lib/ui); при необходимости используйте плагин из подкаталога `plugins`.
