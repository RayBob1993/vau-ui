## Обзор

**VauUI** — набор примитивов и виджетов для построения интерфейсов на Vue 3. Публикуемый npm-пакет — **`vau`** (слои Core и UI доступны из одной точки входа). Исходный код находится в этом же репозитории; документация будет дополняться по мере развития проекта.

### Локальная разработка библиотеки

См. корневой [README](https://github.com/RayBob1993/vau#readme) в репозитории: установка зависимостей, `pnpm serve`, сборка.
